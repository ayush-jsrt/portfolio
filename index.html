Projects

Vidsense

I developed VidSense, a tool designed to deliver concise summaries or answer specific questions from audio and video content. Leveraging the advanced RoBERTa language model for precise and contextually relevant responses, VidSense enhances user engagement by providing quick insights without the need to consume lengthy media. I designed a user-friendly interface for smooth interaction, utilizing the YouTube API to fetch transcripts directly from videos and incorporating RecordRTC for direct audio recording. This project showcases my expertise in AI technologies, web development, API integration, and user interface design, resulting in a technically sophisticated and user-centric tool.

Noida Metro Locator

I developed the Noida Metro Locator (NMRC) using Python, Flask, MySQL, JavaScript, and Leaflet.js to create an interactive metro tracking system. Leveraging Leaflet.js for mapping functionalities, the project provides real-time tracking of Noida Metro, offering a user-friendly interface developed with HTML, CSS, and JavaScript. The backend logic and real-time data APIs are managed using Python, ensuring a seamless experience for commuters. This project employs a robust tech stack, integrating geolocation and push notifications to deliver accurate arrival alerts and enhance commuter convenience.
